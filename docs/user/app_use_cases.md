# Using the App

This document describes common use-cases and scenarios for this App.

## General Usage

Function Codes are managed from the **Function Codes** menu. Each device can have one current Function Code assignment. Existing assignments are updated when a device is assigned to a different Function Code.

## Use-cases and common workflows

- Standardize device purpose labels for operations and automation.
- Filter the Device list by a specific Function Code or by whether a device has any assignment.
- Review assignment completeness from the Coverage dashboard, including breakdowns by status, location, and role.
- Download a CSV template or export current assignments before onboarding or data cleanup.
- Import assignments from a CSV file during onboarding or data cleanup.
- Clear assignments in bulk when device purpose changes or records are retired.
